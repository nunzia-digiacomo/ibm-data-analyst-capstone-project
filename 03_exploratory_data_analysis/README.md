# Exploratory Data Analysis (EDA)

This module explores the dataset to identify trends, correlations, and potential outliers that impact analysis.  

### Key Activities
- Analyzed data distributions using histograms and density plots  
- Detected and removed outliers  
- Calculated feature correlations  
- Summarized findings in a structured DataFrame  

### Tools and Libraries
Pandas, NumPy, Matplotlib, Seaborn  

### Files
- `Finding_How_Data_is_Distributed.ipynb`
- `Finding_Outliers.ipynb`
- `Finding_Correlation.ipynb`
