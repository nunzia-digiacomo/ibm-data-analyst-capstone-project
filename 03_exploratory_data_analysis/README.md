# Exploratory Data Analysis (EDA)

This module explores the dataset to identify trends, correlations, and potential outliers that impact analysis.

### Key Activities
- Analyzed data distributions using histograms and density plots  
- Detected and removed outliers to improve data quality  
- Calculated feature correlations to identify key relationships  
- Summarized insights into a structured DataFrame  

### Tools and Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn
