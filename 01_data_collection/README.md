# Data Collection

This module focuses on collecting and preparing raw data for analysis using APIs and web scraping techniques.  

### Key Activities
- Retrieved job posting data using the GitHub REST API  
- Practiced pagination and JSON data handling  
- Collected and parsed data from HTML pages and tables  
- Exported structured data to CSV for later analysis  

### Tools and Libraries
Python, Requests, BeautifulSoup, Pandas, JSON, CSV
