# Data Wrangling

This module focuses on cleaning, transforming, and preparing datasets for analysis by resolving data quality issues.  

### Key Activities
- Identified and removed duplicate records  
- Detected and imputed missing values  
- Normalized numerical data  
- Ensured dataset consistency and integrity  

### Tools and Libraries
Pandas, NumPy  

### Files
- `Removing_Duplicates.ipynb`
- `Finding_Missing_Values.ipynb`
- `Normalizing_Data.ipynb`
