# IBM Data Analyst Capstone Project

This repository contains my capstone project for the **IBM Data Analyst Professional Certificate**, demonstrating end-to-end data analytics skills ‚Äî from data collection and wrangling to visualization and dashboard creation.  

The goal of this project was to collect real-world data, clean and analyze it using Python and industry-standard tools, and present actionable insights through visualizations and an interactive dashboard.

---

## Project Overview

The project simulates a real-world data analysis scenario.  
It involves collecting job posting data related to data analyst positions and uncovering trends and insights across various technologies and skills.

Through this project, I applied the **data analysis workflow**:
1. **Data Collection** ‚Äî Gathering data from APIs and web scraping sources  
2. **Data Wrangling** ‚Äî Cleaning and normalizing raw data  
3. **Exploratory Data Analysis (EDA)** ‚Äî Detecting trends, correlations, and outliers  
4. **Data Visualization** ‚Äî Creating visual summaries of insights  
5. **Dashboard Creation** ‚Äî Building a professional report for stakeholders  

---

## Tools and Technologies

- **Programming & Analysis:** Python, Pandas, NumPy, Matplotlib, Seaborn  
- **Data Collection:** APIs, Web Scraping (Requests, BeautifulSoup)  
- **Data Visualization:** Matplotlib, Seaborn, Cognos Analytics  
- **Environment:** Jupyter Notebooks, IBM Cognos Analytics, Excel
