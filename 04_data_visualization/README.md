# Data Visualization

This module focuses on visualizing key insights from the dataset using both SQL-based and Python-based techniques.

### Key Activities
- Queried relational databases using SQL to extract data for visualization  
- Created visual summaries using histograms and box plots  
- Built scatterplots and other charts to show relationships between variables  
- Designed data-driven visual reports for effective communication  

### Tools and Libraries
SQL, Python, Pandas, Matplotlib, Seaborn  

### Files
- `Lab - Data Visualization-v1.ipynb`  # SQL + Visualization from RDBMS  
- `Lab 14 - Data Visualization-v1.ipynb`  # Python-based visualizations
