# Data Visualization

This module focuses on visualizing key insights from the dataset using a variety of charts and plots.  

### Key Activities
- Created visual summaries using histograms and box plots  
- Built scatterplots and bubble plots to show relationships between features  
- Designed pie, stacked, line, and bar charts for categorical comparisons  

### Tools and Libraries
Matplotlib, Seaborn, Plotly, Pandas
